What is a domain name?

To put it simply, a domain name is basically internet realty. Domain names regulated by ICANN (The Internet Corporation for Assigned Names and Numbers) - whom I like to think of as the internet's librarians. Domain registrars such as GoDaddy are ICANN accredited sellers of these domain names.
As an example, http://www.google.com is a URL (web address) for Google's main domain name. Google.com would be the domain name. The domain name can be used to describe the main name (part after https://), combined with the domain suffix (or TLD if you are savvy). By domain suffix, we are inferring to .com , .net , .live , .org , .edu , as well as the hundreds of others.
